# 🐶 PetShop Difference

A SaaS platform designed for pet shops that offer recurring subscription plans.

Customers can subscribe to monthly plans, schedule appointments, manage their pets and payments, while administrators manage appointments and services through a dedicated dashboard.

---

## 🚀 Project Goals

- Customer Dashboard
- Pet Registration
- Subscription Plans
- Appointment Scheduling
- Payment Integration
- Admin Dashboard
- Authentication
- REST API

---
