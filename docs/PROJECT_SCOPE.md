# Project Scope

## Overview

PetShop Difference is a SaaS platform for pet shops that want to offer recurring subscription plans for pet care services.

The system will allow customers to subscribe to monthly plans, register pets, schedule appointments, manage payments and track service history.

Administrators will be able to manage customers, pets, appointments, plans and subscription status.

---

## Main Features

- Customer registration
- User authentication
- Pet registration
- Subscription plans
- Appointment scheduling
- Service history
- Payment integration
- Admin dashboard

---

## User Roles

### Customer

- Create account
- Register pets
- Choose a subscription plan
- Schedule appointments
- View service history
- Manage subscription

### Admin

- Manage customers
- Manage pets
- Manage plans
- Manage appointments
- Track subscriptions
- View service history

---

## Out of Scope for MVP

- Mobile app
